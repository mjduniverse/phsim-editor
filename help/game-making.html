Making a Game

Games could be made in the PhSim editor. When this is done, the simulations are treated as levels. Here are the instructions:

  1. Go to the "Edit" menu
  2. Click on "Simulation Settings"
  3. Go to the "Game Settings" tab
  4. Click on "Enable Game"
After doing this, there is a panel that contains three fields:
  • Inital Score: The score the game starts at
  • Goal: The Score that makes one advance levels
  • Life: The life that it starts at
When the desired values have been entered, you can just close the Object Editor. To add a player, one just needs to do the following:
  1. Create an object
  2. Right click on object
  3. Go to main settings
  4. Add at least one sensor class.
  5. Close the Object Editor
To add game health, hazards or coins do the following (please note this only works if the game has been enabled):
  1. Create an object
  2. Right click on object
  3. Open Object Editor
  4. Go to "Widgets" tab
  5. Click on dropdown menu
  6. Click on either "Hazard", "Coin" or "Health"
  7. Click on "Add Widget"
  8. Go to the "Main Settings" tab
  9. Add a censor class that is shared with the player object
